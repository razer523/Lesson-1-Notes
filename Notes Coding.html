Always start your html off with a !DOCTYPE (this tells the browser how to interprupt your code), and open your code body with < html > (and close) When saving your file, save it as either an html, css, or js extension.

A document outline should be as followed: Body -> Header -> Navigation -> Section -> Article -> Footer
Navigation must include < li > and < a > to navigate around site.
Sections have headers.
Articles have headers and footers, or < aside >.

To put a comment into your code, but to not be read on the page, use < ! -- (enter text) - - >

< p > is for paragraphs < /pr >
< br > is for line breaks, no closure needed

HTML Forms- Collecting Info

< form > < /form > Use < input > to collect information from user.
I.e. < input type="text" id="firstName" name="first" />
To add a label use: < label > First Name < /label > before input line.

OR

< label for="firstName"(<- id) > First Name < /label > AND then < input >

PlaceHolders

< input type="text" id="firstName" name="first" placeholder="First Name" />

Validating Form Input Data

Ensuring some information is inputed.
< input name="firstName" required="true" />
OR
< input name="firstName" required />

Input Type valid for validation: "email" "url" "tel" "age"

Validating Value Range

< input name="age" type="number" min="13" max="120" />

Text Area Element

To put a text are (comment field) use: < textarea rows="10" cols="50" name="message" > < /textarea >
